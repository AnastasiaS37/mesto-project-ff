# Mesto – Interactive Web App

Mesto is a study web application that allows users to manage their profile and interact with photo cards.

The project demonstrates working with the DOM, forms, and external APIs.

## 📌 Key Features

* Profile information editing
* User avatar upload
* Uploading and managing personal photos
* Viewing and liking other users’ photo cards
* UI updates without page reload

## 🛠️ Tech Stack

* JavaScript
* HTML5 / CSS3
